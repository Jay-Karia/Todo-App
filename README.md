# Todo App
## A Simple and Cool todo app made from React.js as a learning project 😎

### Functionalities:
 - > Organize your daily tasks in one simple app

 - > Instantly add Todos

 - > 4 different filters are available for your todos

### Features
- Add a todo ➕
- Delete a todo ❌
- Edit a todo ✏
- Mark as done ✅
- Mark as undone ❎
- Add categories to your todos 📁
